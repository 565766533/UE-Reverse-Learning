# UE 游戏逆向学习项目

从指针与内存的基础知识起步，到完成一套可运行的 UE 游戏辅助，本仓库提供 8 个循序渐进的模块与配套代码。新版文档将学习路线、阶段目标与实践任务串联起来，帮助你建立体系化的逆向学习方法。

## 基础与进阶双路线

- [`docs/Foundation.md`](docs/Foundation.md)：聚焦 C/C++、内存模型、工具安装与指针/Pattern Scan 等“打底”能力，配合 `01-BasicMemory` 模块练习。
- [`docs/Advanced.md`](docs/Advanced.md)：串联 UE 对象系统、偏移维护、Hook 技术与 ESP/综合项目实践，对应 `02-08` 模块。

建议按照“基础 → 进阶”的顺序推进，再结合下文的阶段化模块，逐步构建自己的知识库。

## 快速上手

1. 克隆或下载本仓库，阅读 `docs/Syllabus.md` 了解整体节奏。
2. 按照 `docs/ProgressTracker.md` 勾选任务，从阶段 0 的环境预备开始。
3. 每个模块先阅读源码中的注释，再完成大纲里的实践任务与自检。
4. 保留自己的偏移/调试/Hook 笔记，后续模块会频繁复用这些成果。

## 学习阶段概览

| 阶段 | 涵盖模块 | 核心成果 | 建议输出 |
| ---- | -------- | -------- | -------- |
| 阶段 0 ─ 预备 | 环境搭建、知识补齐 | 完成调试环境与工具安装 | 工具安装记录、笔记模版 |
| 阶段 1 ─ 内存基础 | `01-BasicMemory` | 能分析结构体偏移、编写模式扫描伪代码 | 指针链示意图、Pattern Scan 草稿 |
| 阶段 2 ─ UE 数据结构 | `02-UEObjectSystem` + `03-ReverseTools` | 读懂 UE 核心对象、封装常用逆向工具 | 对象访问链图、进程工具脚本 |
| 阶段 3 ─ 工具熟练 | `04-CheatEngine` + `05-x64dbg` | 熟练使用 CE 与 x64dbg | 练习记录、调试日志 |
| 阶段 4 ─ Hook 技术 | `06-HookingTechniques` | 掌握 VTable Hook 与检测规避思路 | Hook 对比表、实验笔记 |
| 阶段 5 ─ 综合实战 | `07-RealWorldProject` + `08-FinalProject` | 完成 ESP + 综合项目，输出复盘 | 功能截图、最终报告 |

更详细的任务拆解与时间建议，请参考 `docs/Syllabus.md`。

## 模块资源

- `01-BasicMemory/`：内存读写、指针、多级指针、特征码扫描基础
- `02-UEObjectSystem/`：模拟 UE4/UE5 的对象系统、名称表、数组容器
- `03-ReverseTools/`：进程/模块操作、内存工具、Pattern Scan 实现
- `04-CheatEngine/`：CE 数值搜索、指针扫描、代码注入练习
- `05-x64dbg/`：断点实践、寄存器观察、调用栈分析、调试技巧
- `06-HookingTechniques/`：VTable Hook Demo、不同 Hook 方案对比
- `07-RealWorldProject/`：模拟 UE 游戏 + ESP 数据采集与渲染
- `08-FinalProject/`：菜单系统、配置管理、多功能整合的完整项目

每个模块的详细任务、检查点与拓展练习均记录在 `docs/Syllabus.md` 中。

## 必备工具

- Visual Studio 2022 Community / C++ 编译环境
- Cheat Engine、x64dbg、IDA Free 或 Ghidra、ReClass.NET（可选）
- Windows 调试环境（推荐使用虚拟机避免影响主机）

工具安装与使用建议已收录在大纲的阶段 0 中，可按步骤完成。

## 学习方法建议

- 坚持“理解 → 实验 → 复盘”的节奏，每个模块都要有自己的总结。
- 结合 CE/x64dbg/IDA 的多工具交叉验证，提高定位偏移的效率与准确度。
- 遇到更新或偏移变化时，先回顾笔记而不是推倒重来，逐渐建立“结构感”。
- 保持法律与道德边界：所有示例仅用于线下学习与技术研究。

祝你在 UE 逆向之旅中保持好奇、持续成长！
